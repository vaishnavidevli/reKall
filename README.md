# REKALL

A storage management and file sharing platform that lets users effortlessly upload, organize, and share files.  
Built with the latest **Next.js 15**, **React 19**, and **Appwrite Node SDK**, it provides a smooth and modern experience for seamless file management.

---

## Features

- **User Authentication with Appwrite**  
  Signup, login, and logout functionality using Appwriteâ€™s authentication system.

- **File Uploads**  
  Upload various file types including documents, images, videos, and audio.

- **View and Manage Files**  
  Browse through uploaded files, view in a new tab, rename, or delete them.

- **Download Files**  
  Download uploaded files for easy offline access.

- **File Sharing**  
  Share uploaded files with others via generated links.

- **Dashboard**  
  View total and used storage, recent uploads, and a summary of files grouped by type.

- **Global Search**  
  Quickly find files or shared content across the platform.

- **Sorting Options**  
  Sort files by name, date, or size for better organization.

- **Responsive Design**  
  A clean and modern UI that looks great on all devices.

---

## Tech Stack

- React 19  
- Next.js 15  
- Appwrite (Node SDK)  
- TailwindCSS  
- ShadCN UI  
- TypeScript
